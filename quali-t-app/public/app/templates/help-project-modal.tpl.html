Help

Edit Project Information

In the first area, you can change the projects information (for example the name).

Edit Quality Attributes

Below the projects information you have the list of all quality attributes.

Add new quality attributes

You can add new quality attributes to the project by using the button. A new modal window will then be opened and you can either select from already created quality attribute templates (a new instance will created for your project then) or you can create a new instance (without variables).

Edit listed quality attributes / Specify variables

To specify the variables of a quality attribute or change the text of a quality attribute click on the icon of the quality attribute to change.

Favorites / Export

Favorite

You can add (or remove) the project to (from) your favorites. This makes the access to a project even faster. Click the icon to toggle the favorites status of the project.

Export

You can export the project as XML or PDF. Click on the icon to achieve this. The generated file should be directly downloaded by your browser.

Validate

After you added and specified the quality attributes of the project, you can check your project for validity. Click the button to achieve this. This means it checks the quality attributes of the project for fuzziness and compares it with other project's values (statistics).

Issue Tracker (JIRA) Export

Export the quality attributes to the Issue Tracker (JIRA) by clicking the button. Before you can export the quality properties, you have to specify the variables (if there are any) and select the quality attributes you want to export (checkbox on the left side of each quality attribute).
