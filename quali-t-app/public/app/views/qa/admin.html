Show all Quality Attribute Templates

Here are all quality attribute templates listed. Each quality attribute template is automatically added to the Standard Catalog. Since these templates can also be linked in catalogs, you have to keep in mind that when you change the template there are some checks:

  • when only options inside variables (ranges, options in dropdowns, default values) change, there will be only a change in the Standard Catalog.
  • when the description itself changes, there will be a new version of the template. This means that every catalog which has this template inside will get the new version of the template in it.
    Projects which have been already created will not get the updated version of the template and remain the same.
